Tipos y variables

En este ejercicio se van a emplear tres variables (x, y, z):
  1. var x (mi nombre)= Nahiara
  2. var y (mi edad)= 18
  3. var z (descripción)= rubia

Con este ejercicio he aprendido a emplear las variables correctamente.He podido observar que al darle un valor a cada una de las variables (x, y, z) el programa directamente las identifica con sus correspondientes valores, en este caso, mi nombre, mi edad y una pequeña descripción mía. De esta manera, cada vez que las emplee, las reconocerá con dichos datos.
