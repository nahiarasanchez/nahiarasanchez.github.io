Web hosting

Un web hosting es lo que en español se podría definir como alojamiento de una web, en el que todos los documentos alojados serán accesibles para cualquier usuario de la web en cualquier momento. Se podría decir que se alquila un espacio de un servidor, que almacena dichos documentos y datos de tu web para que funciones perfectamente.
