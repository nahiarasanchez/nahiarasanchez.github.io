Tipos de selectores que entran en examen

Selector clase

Este tipo de selector se emplea para crear una clase, selecciona las palabras en las que se emplea, y se les cambiará el estilo.

Dentro de la pestaña de estilos, se escribe . seguido de la clase escrita: .clases {...}

Ejemplo:

Selector (class="") clase

Selector identificador

Selecciona únicamente las palabras seleccionadas con el identificador correspondiente. Esto permite cambiar el estilo a dicha palabra

Dentro de la pestaña de estilos, se escribe # seguido del id escrito: #id="" {...}

Ejemplo:

Selector identificador (id="") identificador

Todos los elementos

Este selecciona todos los elementos de, por ejemplo, una lista. Y así cambiar el estilo a todos a la vez.

Dentro de la pestaña de estilos, se escribe *

Ejemplo:

  • clases
  • identificadores
  • todos los elementos

Varios elementos

Este selector nos permite agrupar varios elementos, como un párrafo, una negrita o cursiva... y cambiarles el estilo al mismo tiempo.

Dentro de la pestaña de estilos, se escriben los elementos sin estar separados por comas o, si estan dentro de otro elemento (elemento > elemento): p em strong{...} / div > em {...}

Ejemplo:

Aqui hay elementos.

Elemento contenido dentro de otro elemento

Sirve para seleccionar todos los elementos, de por ejemplo, una order list.

Dentro de la pestaña de estilos, se escriben ambos elementos sin estar separados por comas: ol li

Sobrevuelo

Este selector hace que al pasar el mouse por encima de, por ejemplo, una negrita, esta palabra cambie.

Dentro de la pestaña de estilos se escribe, el elemento que desea que cambie y :holder: strong:holder
